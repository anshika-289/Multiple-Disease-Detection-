# Multiple-Disease-Detection-

This is a Machine Learning Model that is used to predict multiple diseases :
1. Heart Disease 
2. Diabetes
3. Parkinsons

Machine Learning is a great tool to apply in healthcare aiding the doctors and medical staff in early prediction and diagnosis for better and speedy recovery.

# Diabetes Prediction

Uses an Support Vector Machine model to classify if a person has diabetes or not.
Standardization also included.

# Heart Disease Prediction

Uses an Logistic Regression to classify if a person has a heart disease or not.

# Parkinsons Detection

Uses an Support Vector Machine model to classify if a person has parkinsons or not.
Standardization also included.
